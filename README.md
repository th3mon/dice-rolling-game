# Rolling Dice Game

## Prerequisites

- Installed Python v3
- Terminal

## To run game:

```sh
python dice_rolling_game.py
```

## Gameplay example

```sh
python dice_rolling_game.py
Roll the dice? (y/n): y
How many dice you want to roll? (number): 5
(1, 3, 5, 3, 3)
Roll the dice? (y/n): n
Thanks for playing!
```
